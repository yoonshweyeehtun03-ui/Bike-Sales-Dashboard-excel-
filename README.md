# Bike-Sales-Dashboard-excel-
Excel dashboard analyzing bike sales data with key insights on trends, region and occupation
🎯 Objective
This project aims to analyze bike sales data to identify key trends, top-performing products, and regional performance, and to present the findings through an interactive Excel dashboard.
📂 Dataset
The dataset contains sales records with the following key fields:
Product
Region
Sales / Revenue
Income
Education
Martial Status
⚙️ Tools Used
Microsoft Excel
Pivot Tables
Data Cleaning
Charts and Visualizations
Slicers / Filters
📊 Dashboard Features
Interactive filters for dynamic analysis
Sales performance by product and region
Clear visual summaries using charts
🧠 Conclusion
The dashboard provides a clear overview of bike sales performance and helps identify important business insights that can support decision-making.
